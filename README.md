# Employee-Tracker

## Description 

A node database app to help you manage your employees.

## Table of Contents (Optional)

* [Installation](#installation)
* [Usage](#usage)

## Installation

requires node.js & mysql.

to install, write in console inside employee-tracker directory:

npm install

## Usage 

to run:

node server.js
